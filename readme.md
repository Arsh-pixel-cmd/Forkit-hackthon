# O'FOOD - Gastronomy Trust Engine

This is a Next.js 15 application built with Tailwind CSS and the App Router.

## Features

- **Food Trust Engine**: Verify meal quality with 5-photo upload.
- **Deal Finder**: Compare prices across food delivery apps.
- **Glassmorphism UI**: Beautiful, soft UI.
- **Mock API**: `/api/audit-dish` for verification logic.

## Getting Started

1.  Wait for dependencies to finish installing (or run `npm install` manually).
2.  Run the development server:

```bash
npm run dev
```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser.

## Structure

- `/app`: App Router pages and API routes.
- `/components/glassmorphism`: UI Components.
- `/lib/foodoscope`: API wrappers/types.
